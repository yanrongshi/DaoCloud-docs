## Verification

In the directory where the offline installation package and verification file are downloaded, take v0.13.1_amd64 as an example, run the following command to verify the integrity:

```sh
echo "$(cat insight_v0.13.1_amd64_checksum.sha512sum)" | sha512sum -c
```

After the verification is successful, the printed result is similar to:

```none
insight_v0.13.1_amd64.tar: ok
```
