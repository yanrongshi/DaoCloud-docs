# 虚拟机容器 Release Notes

本页列出虚拟机容器的 Release Notes，便于您了解各版本的演进路径和特性变化。

## 2023-8-31

### v0.2.0

#### 新功能

- **新增** 支持重启虚拟机
- **新增** 支持表单编辑虚拟机
- **新增** 支持 YAML 创建虚拟机
- **新增** 支持克隆虚拟机
- **新增** 支持为虚拟机创建快照
- **新增** 支持从快照恢复虚拟机，并展示恢复记录

## 2023-7-31

### v0.1.0

#### 新功能

- **新增** 支持通过集群展示虚拟机列表
- **新增** 支持通过容器镜像创建虚拟机
- **新增** 支持关机/启动虚拟机
- **新增** 支持删除虚拟机
- **新增** 支持通过控制台访问（VNC）虚拟机
- **新增** 支持查看虚拟机详情
